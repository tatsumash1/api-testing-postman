# API-тестирование в Postman

Портфолио-проект по тестированию REST API сервиса ReqRes.

## Цель проекта

Показать базовые навыки API-тестирования:

- анализ API-документации;
- составление тест-плана;
- подготовка чек-листа;
- написание тест-кейсов;
- выполнение позитивных и негативных проверок;
- оформление баг-репортов;
- работа с Postman Collection;
- использование переменных окружения;
- подготовка тестового отчета.

## Тестируемый сервис

- Сервис: ReqRes
- API: The classic API
- Base URL: `https://reqres.in`

## Что будет внутри

В этом проекте будут описаны проверки API ReqRes: получение пользователей, создание пользователя, обновление данных, удаление пользователя, регистрация, логин и негативные сценарии.

В коллекции использовались проверки через `pm.test`: статус-коды, наличие обязательных полей, типы данных и структура JSON-ответов.

## Запуск коллекции

Все запросы были объединены в Postman Collection и запущены через Collection Runner. Это позволяет выполнять весь набор API-проверок одним запуском и быстро повторять регрессионную проверку.
