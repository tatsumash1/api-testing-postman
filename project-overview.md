# Описание проекта

## Название проекта

ReqRes API Testing

## Краткое описание

Этот проект посвящен тестированию REST API сервиса ReqRes через Postman.

## Цель тестирования

Проверить, что основные API-методы работают корректно и возвращают ожидаемые статус-коды, структуру JSON-ответа и данные.

## Инструменты

- Postman
- ReqRes Classic API
- Markdown-документация
- GitHub

## Область тестирования

Будут проверяться следующие группы запросов:

- пользователи;
- создание пользователя;
- обновление пользователя;
- удаление пользователя;
- регистрация;
- авторизация;
- негативные сценарии.
