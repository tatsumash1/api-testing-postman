# Чек-лист

## Общие проверки

- [ ] Base URL указан через переменную окружения.
- [ ] Все запросы сохранены в Postman Collection.
- [ ] У каждого запроса есть понятное название.
- [ ] У запросов указаны корректные HTTP-методы.
- [ ] Для POST, PUT и PATCH запросов заполнен request body.
- [ ] Для каждого запроса проверен status code.
- [ ] Для JSON-ответов проверена структура.
- [ ] Проверены обязательные поля в ответе.
- [ ] Проверены позитивные сценарии.
- [ ] Проверены негативные сценарии.

## Users

- [ ] Получение списка пользователей.
- [ ] Получение одного пользователя.
- [ ] Проверка несуществующего пользователя.
- [ ] Проверка пагинации.

## Create / Update / Delete

- [ ] Создание пользователя.
- [ ] Полное обновление пользователя.
- [ ] Частичное обновление пользователя.
- [ ] Удаление пользователя.

## Register / Login

- [ ] Успешная регистрация.
- [ ] Регистрация без пароля.
- [ ] Успешный логин.
- [ ] Логин без пароля.

## Отчетность

- [ ] Результаты тестирования описаны в отчете.
- [ ] Баги оформлены в отдельном файле.
- [ ] Скриншоты добавлены при необходимости.
